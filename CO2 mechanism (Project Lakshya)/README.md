# CO2 Mechanism (Project Lakshya)

CO2-based pyrotechnic-free ejection system designed as the primary recovery mechanism 
for Project Lakshya, a student-built sound rocket.

## Overview
This mechanism handles parachute deployment at apogee using a CO2 cartridge-based 
ejection charge, avoiding traditional pyrotechnic ejection. The design prioritizes 
reliability and repeatability for recovery, critical for a competition flight where 
a failed deployment means losing the vehicle.

## My Role
Designed the CO2 ejection mechanism — from concept through CAD modeling and 
mechanical validation, as part of the recovery subsystem for Project Lakshya.

## Result
 **1st Prize at InSpace** — Project Lakshya reached a peak altitude (apogee) of 
**998 meters**, with successful recovery using this mechanism.
